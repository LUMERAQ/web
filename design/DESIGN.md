# Instrucciones Finales para la IA

Genera una página web corporativa de nivel internacional para **LUMERAQ**, con calidad visual comparable a Apple, Stripe, OpenAI, Vercel y Microsoft. El resultado debe ser completamente responsive, accesible, optimizado para SEO y rendimiento, con una experiencia de usuario premium, animaciones fluidas, diseño minimalista y enfoque en conversión. Mantén estrictamente la identidad visual definida (paleta azul/cian/morado, tipografía Poppins, estilo oscuro elegante, iconografía outline y fotografías corporativas reales). La web debe transmitir confianza, innovación y capacidad tecnológica empresarial desde el primer segundo.
